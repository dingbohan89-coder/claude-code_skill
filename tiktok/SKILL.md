# TikTok Video Scraper

此技能用于抓取 TikTok 视频信息。

## 使用方法

在对话中直接说类似以下的话：
- "帮我抓取这个 TikTok 视频的信息: https://www.tiktok.com/@username/video/123456"
- "获取这个 TikTok 视频的统计数据"

## 功能

- 抓取视频标题、描述
- 获取作者信息
- 获取互动数据（点赞、评论、分享、收藏）
- 提取相关标签

## 首次使用

首次使用前需要登录 TikTok 账号：
```bash
node C:/Users/hp/claude_skills/claude_skills/tiktok-skill/scripts/login.js
```
